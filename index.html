Routes List

/register

  • POST - Register a new user

/login

  • POST - Log in with existing user credentials

/task-status

  • GET - Get all task statuses
  • POST - Insert a new task status
  • PUT - Update a task status by ID
  • DELETE - Delete a task status by ID

/task

  • GET - Get all tasks
  • POST - Insert a new task
  • PUT - Update a task by ID
  • DELETE - Delete a task by ID

/project

  • GET - Get all projects
  • POST - Insert a new project (authentication required)
  • PUT - Update a project by ID
  • DELETE - Delete a project by ID
